# Project Overview

This project is a microservices-based application built with **Spring Boot** and **Maven**. It includes:
- **User Service**: Handles user registration, authentication, and management.
- **Listing Service**: Manages listings (e.g., products, items, etc.).
- **Security**: Uses JWT for authentication and authorization.
- **Database**: MySQL with Liquibase for migrations.
- **API Gateway**: (DONE) Set up using Spring Cloud Gateway.

## Technologies Used

- Java, Spring Boot
- Maven
- MySQL
- Liquibase
- JWT (JSON Web Token)
- Python (for auxiliary scripts)
- IntelliJ IDEA

## Getting Started

### Prerequisites

- Java 17+
- Maven 3.8+
- MySQL 8+
- Python 3.x (optional)
